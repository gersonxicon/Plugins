Description: This plugin shows an overlay panel in 'infopanel' section.
Administrators can send messages to all users in the space using the 'Send Info' menu; using 'People' menu
they can send messages to an specific person as well, all these messages will be shown in the overlay panel.
The other option is to link an object in the space, so when users walk inside that object or zone, an 
alert will be shown and a message will be placed in the overlay panel.
The overlay panel can be closed or open using 'Hide/Show Panel' menu.

Parameters:
-txt-onEnter: Title for message when user enters to an object.
-txt-firstMessage: Text to show on first section of overlay panel.
-txt-secondMessage: Text to show on second section of overlay panel.
