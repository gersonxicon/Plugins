Description: Shows an alert and teleport users based on a key combination (Ctrl+I).

Parameters:
-Enabled: When enabled, the popup will be shown.
-Sound: When enabled, sound will be available on teleport.
-Title: Title of the message.
-Text: Text to display in the message.
-Button Text: Text to display in the OK button.
-X Location: Set X location to transport.
-Y Location: Set Y location to transport.
-Z Location: Set Z location to transport.
-Radius: Set radius to teleport (integer).
