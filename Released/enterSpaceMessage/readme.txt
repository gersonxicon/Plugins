Description: This plugin provides the capability to add an entrance message, this popup will be shown
for each user.  
This message will ask users if they want to be transported to a defined location in the space or not.

Parameters:
-Enabled: When checked, the popup will be shown.
-Sound: When enabled, sound will be available on teleport.
-Title: Title of the message.
-Text: Text to display in the message.
-Yes Button Text: Text to display in Yes Button.
-No Button Text: Text to display in No Button.
-X Location: Set X location to transport on affirmative response.
-Y Location: Set Y location to transport on affirmative response.
-Z Location: Set Z location to transport on affirmative response.
-Radius: Set radius to teleport (integer).
-Negative Response Text: Text to display on negative response.
